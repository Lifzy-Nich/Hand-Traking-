🎮 3D Particle System

Hand Gesture Control

Sistem partikel 3D interaktif yang dikendalikan oleh gestur tangan Anda secara real-time.

Klik tombol di bawah untuk memulai. Browser akan meminta izin akses kamera.

Memuat library...

⚠️ Penting: Jika popup kamera tidak muncul, pastikan Anda membuka halaman ini melalui localhost atau HTTPS.

Cara menjalankan via Local Server:
# Python 3 (paling mudah):
python -m http.server 8000

# Node.js:
npx serve .

# PHP:
php -S localhost:8000

# Lalu buka: http://localhost:8000
